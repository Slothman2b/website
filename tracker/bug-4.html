    Rhino Linux | Firefox not displaying landing page on AMD64 devices.

    Due to us utilising a different package for Firefox on AMD64 devices, the Firefox landing page was not set on AMD64 devices, which coincidentally is our most popular platform at the moment. To fix this issue we have updated our Firefox binary pacscript place firefox files in /etc/firefox instead of /opt/firefox, which should mitigate this issue. To have our firefox landing page correctly load on AMD64 platforms please run:

    rpk update -y and then

    echo 'pref("browser.startup.homepage", "https://rhinolinux.org/landing/");' | sudo tee -a /lib/firefox/defaults/pref/syspref.js
